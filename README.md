# Calculator Application with Error Handling & Logging

## Description
This project is a Python calculator application that performs basic arithmetic operations with proper exception handling and logging.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for:
  - Invalid input
  - Division by zero
  - Unexpected runtime errors
- Logging using Python logging module

## Files Included
calculator.py - Main calculator program
calculator.log - Log file generated automatically
README.md - Project documentation

## Requirements
Python 3.x

## How to Run
1. Open terminal or command prompt
2. Navigate to project folder
3. Run:

python calculator.py

## Logging
All operations and errors are saved in calculator.log file.
